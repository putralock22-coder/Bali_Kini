Bali Kini CMS

Pilih artikel dari sidebar untuk mengedit, atau klik ＋ Artikel Baru untuk membuat artikel baru.

Semua perubahan disimpan langsung ke GitHub.

⚙ Pengaturan GitHub

Masukkan GitHub Personal Access Token dengan scope repo untuk mengakses dan mengedit konten.

Settings → Developer settings → Personal access tokens → Tokens (classic) → scope: repo
🌐 Terjemahan Otomatis (Opsional)
Dapatkan di console.anthropic.com → API Keys
Default: localhost:11434
⚠ Aktifkan CORS Ollama sebelum pakai.
Jalankan Ollama dengan perintah di terminal:
set OLLAMA_ORIGINS=* && ollama serve Atau tambah di environment variable sistem: OLLAMA_ORIGINS=*
